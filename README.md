# Malaysia Postcodes

A huge list of Malaysia postcodes in a single file: [ALL-POSTCODES.csv](ALL-POSTCODES.csv)

Postcodes are displayed in **CSV** ("Comma Separated Values") in these following formats:-
```
"POSTCODE","ADDRESS","CITY","STATE"
```
